# MyDogCrush

Proyecto base de una app Android tipo Tinder para perros.

## Qué incluye
- Pantalla principal con tarjeta de perro
- Botones de Me gusta y Pasar
- Estilo visual inspirado en Tinder
- Estructura lista para abrir en Android Studio

## Cómo abrirlo
1. Abre Android Studio
2. Selecciona Open
3. Elige la carpeta de este proyecto
4. Espera a que sincronice Gradle
5. Ejecuta la app en un emulador o dispositivo
